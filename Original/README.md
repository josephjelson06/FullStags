# UrgentParts — Emergency Industrial Parts Matching

Intelligent B2B order allocation platform that matches emergency part orders with the nearest available supplier, computing optimized delivery routes in real time.

## Quick Start

```bash
# Backend (Terminal 1)
cd backend
pip install -r requirements.txt
python run.py          # → http://localhost:8000

# Frontend (Terminal 2)
cd frontend
npm install
npm run dev            # → http://localhost:5173
```
